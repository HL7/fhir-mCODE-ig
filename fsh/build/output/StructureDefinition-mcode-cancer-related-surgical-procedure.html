StructureDefinition: CancerRelatedSurgicalProcedure

A surgical action addressing a cancer condition. The scope of this profile has been narrowed to cancer-related procedures by constraining the ReasonReference and ReasonCode to cancer conditions. Conformance note: If an ICD-10-PCS code is used in the code attribute, and there is a semantically equivalent SNOMED CT or CPT code, the resulting Procedure instance will not be compliant with US Core Profiles.

The official URL for this profile is: 

http://hl7.org/fhir/us/mcode/StructureDefinition/mcode-cancer-related-surgical-procedure
